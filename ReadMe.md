#  elmer-common-ui 基于elmer-ui-core框架编写的组件库

### 支持组件
    * eui-alert: 自定义提示框
    * eui-button: 按钮，支持定义Icon
    * eui-calendar: 日历控件
    * eui-dialog: 弹窗
    * eui-drop-down: 下拉框
    * eui-form: 表单
    * eui-form-item: 表单项
    * eui-icon: 图标组件
    * eui-icon-demo: 所有图标列表
    * eui-mobile-select: 手机端下拉框
    * eui-qqservice: QQ在线客服
    * eui-tab:  Tab选项卡
    * eui-tab-item: 选项卡项
    * eui-toast: toast提示
    * eui-upload: 文件长传
